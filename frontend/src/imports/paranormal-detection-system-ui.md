Design a mobile web app interface for a ghost-hunting game called “Paranormal Detection System.”

This is not a native mobile app. It is a mobile website built with React that will run in a phone browser and be deployed on the web. The design should be optimized for mobile screens (375–420px width).

The interface should feel like a futuristic ghost detection device, similar to a radar scanner used to track paranormal activity.

STYLE:

Dark cyberpunk / hacker theme

Background color: black or very dark (#0a0a0a)

Primary color: neon green (#00ff9c)

Warning color: red (#ff3b3b)

Monospace / terminal style typography

Minimal but immersive UI

Glowing neon effects for radar and important elements

The app should have three screens.

SCREEN 1 – SYSTEM BOOT / LOADING SCREEN
When the app opens, show a system initialization screen that looks like a device starting up.

Elements:

Large title: PARANORMAL DETECTION SYSTEM

Terminal style messages such as:

Initializing paranormal scanner

Loading radar module

Calibrating sensors

Establishing ghost detection network

System ready

A loading progress bar or animated loading indicator

Blinking cursor or terminal-style animation

The screen should feel like a ghost detection device booting up.

SCREEN 2 – AGENT LOGIN
After loading, show a simple login screen.

Elements:

Title: ENTER AGENT NAME

Text input field for the player name

Large button: START HUNT

Clean minimal layout centered on screen

Same dark futuristic theme

SCREEN 3 – MAIN GHOST DETECTOR SCREEN
This is the main gameplay screen where players search for ghosts in a room.

Layout from top to bottom:

Header

Title: PARANORMAL DETECTION SYSTEM

Small subtitle: Ghost Hunter Radar

Status indicator: System Status: Active

Radar Section

Large circular radar in the center

Radar sweep animation rotating around the circle

Grid lines inside radar

Neon green glow effect

Signal Strength Section

Label: Signal Strength

Visual signal bar indicator or strength meter

Levels:

No Signal

Faint Signal

Strong Signal

VERY STRONG SIGNAL

When signal is very strong, show a floating ghost icon near the radar

Ghost Indicator

Ghost icon (👻) appears when the ghost is close

Slight floating animation

Main Action Button

Large button below radar

Text: CAPTURE GHOST

Bright neon green button

Rounded corners

Mobile-friendly tap size

Terminal Log Panel
At the bottom of the screen show a terminal-style message panel displaying messages such as:

RADAR ACTIVE

SCANNING ROOM

PARANORMAL SIGNAL DETECTED

This panel should look like a system console.

UX CONCEPT
The player walks around a room with their phone.
The radar detects ghosts and shows signal strength.
When the signal becomes very strong, the player presses CAPTURE GHOST to catch the ghost.

The design should feel like a ghost detection gadget interface used by paranormal investigators.

Focus on a clean single-page mobile interface with a strong radar visual and clear capture interaction.
